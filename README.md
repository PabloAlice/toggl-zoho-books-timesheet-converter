Convert Toggl timesheet CSV to Zoho Books format

# Installation
```
npm install toggl-zoho-books-timesheet-converter
```

# Usage
```
node index.js Toggl_time_entries_2016-08-01_to_2016-08-31\ 11.59.43\ a.csv > out.csv
```

# Preparing CSV
- "Detailed" view
- Rounding on
- Download as CSV
